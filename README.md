개요
---------------------------------------------------------------
chainpool gateway(Node.JS 웹서버) 프로젝트

의존성
---------------------------------------------------------------
Install Node.JS, nodemon

```
curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo npm install nodemon -g
```

실행
---------------------------------------------------------------
* node modules 설치
<pre>
	npm install
</pre>
* 서버 시작
<pre>
	bash chainpool.sh
</pre>
