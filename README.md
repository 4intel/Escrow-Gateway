Summary
---------------------------------------------------------------
Escrow Gateway(Node.JS Web Server) Project

Prerequisite
---------------------------------------------------------------
Install Node.JS, nodemon

```
curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo npm install nodemon -g
```

Usage
---------------------------------------------------------------
* Install node modules
<pre>
	npm install
</pre>
* Start Server
<pre>
	bash chainpool.sh
</pre>
